# KeyboardPet 发布说明

## 适用系统

- Windows 10 / Windows 11

## 快速使用

1. 双击 `KeyboardPet.exe`
2. 桌面右下角会出现熊猫桌宠
3. 右键熊猫或使用系统托盘图标打开菜单

## 主要功能

- 跟随键盘敲击和鼠标点击动作
- 眨眼、犯困、悬停挥手、摸头互动
- 喝水提醒和起身活动提醒
- 摸鱼背单词
- 托盘菜单
- 开机启动
- 音效开关
- 鼠标避让

## 说明

- 程序是单实例的，重复打开不会出现多只熊猫
- 程序不会读取或保存你输入的文字内容
- 设置保存在：

```text
%APPDATA%\KeyboardPet\settings.json
```

## 首次运行可能遇到的问题

- 如果 Windows Defender 弹出提示，点击“更多信息”后选择“仍要运行”
- 如果位置异常，删除设置文件后重新启动
