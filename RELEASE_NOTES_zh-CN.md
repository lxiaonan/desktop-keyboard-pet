# KeyboardPet 发布说明

## 适用系统

- Windows 10 / Windows 11

## 快速使用

1. 双击 `KeyboardPet.exe`
2. 桌面右下角会出现熊猫桌宠
3. 右键熊猫或点击系统托盘里的熊猫图标打开菜单

打包好的 `KeyboardPet.exe` 已经包含熊猫图片和动画帧，普通用户不需要安装 Python。背单词功能需要用户自己创建本地词库文件。

## 主要功能

- 跟随键盘敲击和鼠标点击动作
- 眨眼、犯困、悬停挥手、摸头互动
- 喝水提醒和起身活动提醒
- 摸鱼背单词
- 可按格式创建本地词库，用于摸鱼背单词
- 心情、体力、清洁、饱腹、亲密度、等级养成
- 喂零食、洗澡、休息互动
- 托盘菜单
- 开机启动
- 音效开关
- 鼠标避让

## 常用入口

- 右键熊猫：打开常用菜单
- 托盘图标：显示/隐藏、退出、开机启动、音效、提醒、背单词
- 熊猫状态：查看养成数值，进行喂食、洗澡、休息
- 摸鱼背单词：选择词库，标记认识/不认识

## 说明

- 程序是单实例的，重复打开不会出现多只熊猫
- 程序不会读取或保存你输入的文字内容
- 背单词进度和萌宠设置只保存在本机
- 词库文件需要用户自行创建，路径是 `%APPDATA%\KeyboardPet\word_banks.json`
- 设置保存在：

```text
%APPDATA%\KeyboardPet\settings.json
```

## 首次运行可能遇到的问题

- 如果 Windows Defender 弹出提示，点击“更多信息”后选择“仍要运行”
- 如果位置异常，删除设置文件后重新启动
